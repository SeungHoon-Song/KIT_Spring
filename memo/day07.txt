전자정부표준프레임워크(egov)
	정부 공공 프로젝트 등 상용화 혹은 서비스해야 할 상품을 제작할 때
	표준 라이브러리만 제공해준다.
-----------------------------------------------------------------------------------------
엔드 : 서버

웹 디자이너 : 포토샵, 일러스트 파일로 웹 개발에 필요한 요소 디자인
프론트 엔드 : 백에서 요청한 뷰를 알맞게 응답
퍼블리셔 : 프론트 엔드를 도와주는 직종 (한국에만 있는 직종)
백 엔드 : REST
풀스택 : 백 + 프론트 + 엔드

1. 웹 디자인 포토샵 파일 완성
2. 프론트 엔드 개발자가 각 요소를 디자인 이미지에서 분리
3. 백 엔드 개발자가 해당 이미지를 사용

클라이언트 --> 백 단 서버 --> 연산, 동적데이터 처리
--> 프론트 요청 --> 프론트 응답 --> 백 단 응답
-----------------------------------------------------------------------------------------
